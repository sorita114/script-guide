* Overview

웹 페이지 작성시 필요한 동적요소와 Interaction 요소에 대해 웹접근성을 준수하기 위한 Javascript 가이드를 제공합니다.

가이드 항목 : Layer popup, Rolling Contents, Form Validate, DropDown

* 본 문서는 bootstrap 으로 작성되었으며, 별도의 서버구성 없이 내용을 확인할 수 있습니다.

사용되는 파일
jquery.js (1.7.2+)
jquery.ui.js
jquery.package.js
jquery.validate.js

Global 참조 파일

<script type="text/javascript" src="../js/front/jquery/jquery.js"></script>
<script type="text/javascript" src="../js/front/jquery/jquery-ui/jquery-ui.js"></script>
<script type="text/javascript" src="../js/front/jquery/plugins/jquery.package.js"></script>

Locale 참조 파일

<script type="text/javascript" src="../js/front/jquery-validation/jquery.validate.js"></script> <!-- jquery validate 플러그인 파일. -->
<script type="text/javascript" src="../js/front/jquery-validation/jquery.validate.init.js"></script> <!-- initialize 파일 : 경고 문구, 기본 설정 셋팅. -->

* Layer Popup

Layer Popup 은<script type="text/template">...</script> 으로 구성된 Template 요소가 필요하다.

<script type="text/javascript" src="../statics/scripts/jquery/jquery.js"></script>
<script type="text/javascript" src="../statics/scripts/jquery/plugins/jquery.package.js"></script>

의존성 : jquery.js (1.7.2+), jquery.package.js, template element (text/template)

코드


<button type="button" data-trigger="layerpopup" data-target="#tpl_layer">레이어팝업 오픈</button>
<script type="text/template" id="tpl_layer">
<div class="wrap-layerpop">
    <div class="sect-layerpop">
        <div class="layerpop-contaniner">
            <h2>레이어 타이틀</h2>
            <div class="layerpop-contents"></div>
        </div>
        <button class="btn-close">닫기</button>
    </div>
</div>
</script>

* Form Validation

Validation Check 가 필요한 경우, jquery.validate.js, jquery.validate.init.js 파일을 추가한다.

<script type="text/javascript" src="../statics/scripts/jquery/jquery.js"></script>
<script type="text/javascript" src="../jquery.validate.js"></script> <!-- jquery validate 플러그인 파일. -->
<script type="text/javascript" src="../jquery.validate.init.js"></script> <!-- initialize 파일 : 경고 문구, 기본 설정 셋팅. -->

Check 조건은 각 input 의 attribute에 작성한다.

$('form').validate(); 를 실행한다.

의존성 : jquery.js (1.7.2+), jquery.validate.js (1.13.1), jquery.validate.init.js

input 요소에 name 값은 필수입니다.

코드


<form>
	<input type="text" id="name" name="name" title="이름" required="required">
	<input type="text" id="email" name="email" title="이메일" required="required" email="email">
	<button type="submit">전송</button>
</form>
<script type="text/javascript" src="../jquery.validate.js"></script>
<script type="text/javascript" src="../jquery.validate.init.js"></script>
<script type="text/javascript">
	$('form').validate();
</script>
