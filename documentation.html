* Overview

웹 페이지 작성시 필요한 동적요소와 Interaction 요소에 대해 웹접근성을 준수하기 위한 Javascript 가이드를 제공합니다.

가이드 항목 : Layer popup, Rolling Contents, Form Validate, DropDown

* 본 문서는 bootstrap 으로 작성되었으며, 별도의 서버구성 없이 내용을 확인할 수 있습니다.

* Layer Popup

Layer Popup 은<script type="text/template">...</script> 으로 구성된 Template 요소를 필요로 한다.

이벤트 등록은 일괄, 개별 두가지 방법으로 가능하다.

의존성 : jquery.js (1.7.2+), jquery.package.js, template element (text/template)

data-target (필수)

data-target 값을 <script type="text/template"> 의 id 앞에 '#'을 붙여 작성한다.


<button type="button" data-trigger="layerpopup" data-target="#[TEMPLATE ID]">레이어팝업 오픈</button>

template (필수)

Layer Popup Template 은 아래 작성된 wrap-layerpopup, sect-layerpopup, layerpopup-container, h2, layerpopup-contents button 를 필수요소로 가진다.

btn-close 클래스명은 Layer Popup 을 닫는 trigger 로 사용된다.

template 영역에서 button, a Tag 에 class="btn-close" 를 추가하면 자동으로 Layer Popup 이 닫힌다.


<script id="tpl_layer" type="text/template">
<div class="wrap-layerpop">
    <div class="sect-layerpop">
        <div class="layerpop-container">
            <h2>레이어 타이틀</h2>
            <div class="layerpop-contents"></div>
        </div>
        <button class="btn-close">닫기</button>
    </div>
</div>
</script>

data-trigger (옵션)

data-trigger 는 전역에서 초기화 될 때, layerpopup 이벤트를 발생시키기 위한 조건으로 사용된다.


<button type="button" data-trigger="layerpopup" data-target="#[TEMPLATE ID]">레이어팝업 오픈</button>

개별로 이벤트를 핸들링해야하는 경우, data-trigger 를 제거하고 trigger 가 되는 Element 를 대상으로 jquery plugin 함수인 layerpopup 을 실행하면 된다.

callback 옵션에서 전달받는 인자는 생성되는 Layer Popup 의 instance 로 jQuery DOM 타입이다.


$('#btnLayer').layerpopup({
	callback : function ($layer) {
		// $layer : layer DOM instance (jQuery DOM)
		// $layer.append or $layer.html 작성가능.
	}
});

* Rolling

indecator는 각 item 블럭에 넣어준다.

thumbnail 을 사용할 경우, slide-wrap 안에 thumbnail-wrap 블럭을 추가한다.

<div class="item-thumbnail-wrap">
    <ul>
        <li>
            <a href="#"><img src="[ 썸네일 이미지 url ]" alt="[ 썸네일 이미지 설명 ]"/></a>
        </li>
        <li>
            <a href="#"><img src="[ 썸네일 이미지 url ]" alt="[ 썸네일 이미지 설명 ]"/></a>
        </li>
        <li>
            <a href="#"><img src="[ 썸네일 이미지 url ]" alt="[ 썸네일 이미지 설명 ]"/></a>
        </li>
    </ul>
</div>

의존성 : jquery.js (v1.7.2+), jquery.ui.js( v1.11.4 ), jquery.package.js

term ( 선택 )

defaults
4000

슬라이드가 되는 시간


$('#mainNoticeSlider').visualMotion({
	term : 4000
});

roof ( 선택 )

defaults
true

마지막 이미지에서 처음 이미지를 호출한다.


$('#mainNoticeSlider').visualMotion({
	roof : true
});

auto ( 선택 )

defaults
true

컨텐츠 로드 시 슬라이드가 된다.


$('#mainNoticeSlider').visualMotion({
	auto : true
});

* Form Validate

Validation Check 가 필요한 경우, jquery.validate.min.js, jquery.validate.init.js 파일을 추가한다.

<script type="text/javascript" src="../jquery.validate.min.js"></script> <!-- jquery validate 플러그인 파일. -->
<script type="text/javascript" src="../jquery.validate.init.js"></script> <!-- initialize 파일 : 경고 문구, 기본 설정 셋팅. -->

Check 조건은 각 input 의 attribute에 작성한다.

$('form').validate(); 를 실행한다.

의존성 : jquery.js (1.7.2+), jquery.validate.js (1.13.1)

input 요소에 name 값은 필수입니다.

Validate Check 조건


<!-- required : 필수 입력 항목 -->
<input type="text" required="required" />

<!-- email : 이메일 형식 (aaa@domain.com) -->
<input type="text" email="email" />

<!-- url : 웹페이지 주소 형식 (htttp://www.domain.com) -->
<input type="text" url="url" />

<!-- date : 날짜 형식 체크 -->
<input type="text" date="date" />

<!-- number : 실수 형식 체크 (ex : 1, 1.3) -->
<input type="text" number="number" />

<!-- digits : 자연수 형식 체크 -->
<input type="text" digits="digits" />

<!-- creditcard : 카드번호 형식 체크 -->
<input type="text" creditcard="creditcard" />

<!-- engnum : 영문, 숫자 허용 -->
<input type="text" engnum="engnum" />

<!-- equalTo : 값 비교 id selector 지정 -->
<input type="text" equalTo="#{Element ID}" />

<!-- maxlength : 최대 글자수 -->
<input type="text" maxlength="3" />

<!-- minlength : 최소 글자수 -->
<input type="text" minlength="2" />

<!-- length : 글자수 지정 -->
<input type="text" length="3" />

<!-- rangelength : 글자수 범위지정 -->
<input type="text" rangelength="3,5" />

<!-- range : 입력 숫자 범위 지정 -->
<input type="text" range="3,5" />

<!-- max : 입력 숫자 최대값 -->
<input type="text" max="5" />

<!-- min : 입력 숫자 최소값 -->
<input type="text" min="2" />

적용

$.fn.validate

form submit 발생시 validate 가 동작한다.


$('form').validate();

$.fn.valid

submit 과 상관없이 validate 체크가 필요한 경우 사용한다.

submit 하기 전에 사전체크가 필요하거나 비동기로 submit 을 처리하는 경우 사용할 수 있다.


var isSuccess = $('form').valid();
if(iskSuccess){
	// 성공 처리.
}

* Dropdown

의존성 : jquery.js (1.7.2+), jquery.package.js

onChange ( 선택 )

해당 모듈로 값을 변경하는 경우 사용. 


<script type="text/javascript">
$(".filter").dropdown({
	onChange : function( value ){
		// change value.
	}
});
</script>

data-value : onChange 이벤트 발생시 전달인자 값.


<li><a href="#" data-value="0">전체</a></li>
